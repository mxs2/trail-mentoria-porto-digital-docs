# Projeto Trail — Mentoria Porto Digital

Este repositório centraliza **toda a documentação da mentoria do Projeto Trail**, realizada no contexto do Programa Residência Porto Digital, com mentoria técnica da Avanade.

## 🎯 Objetivo

Guiar os alunos na construção de um **MVP full stack profissional**, passando por:

- Discovery de produto
- Modelagem de domínio
- Arquitetura
- Métricas
- Segurança
- Deploy em cloud

Este repositório **não contém código**.  
O código-fonte do projeto será mantido em um repositório separado.

---

## 📚 O que você encontrará aqui

- Documentação de Discovery
- Planejamento e materiais de cada encontro
- Atividades e desafios técnicos
- Registros de decisões arquiteturais
- Referências e boas práticas
